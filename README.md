# mme_analyzer

Custom Frappe / ERPNext app for the MME analyzer support platform.

## Purpose
This app extends ERPNext and Marley with analyzer-focused support workflows.

## Phase 1 modules
- Analyzer Registry
- Customer Site
- QC Log
- Calibration Log
- Maintenance Log
- Alert Rule
- Support Incident

## Planned ERPNext screens
- Fleet Dashboard
- Analyzer List and Form
- Customer Site List and Form
- QC Log List
- Calibration Log List
- Maintenance Log List
- Support Incident List

## Notes
This is an initial scaffold created for development planning.
It is meant to be installed as a separate custom app on top of Frappe, ERPNext, and Marley.
